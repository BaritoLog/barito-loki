# Barito Loki

Pushing logs to Grafana Loki, through POST requests with Barito Router's context format. 

## Development Setup 

Fetch and build the project.
```sh
git clone https://github.com/vwidjaya/barito-loki
cd barito-loki
go build
```

## Run Instructions
After the project is built, run:
```sh
./barito-loki
```
